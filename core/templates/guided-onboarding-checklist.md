# ClickUp Operator Guided Onboarding Checklist

## Connection

- official ClickUp MCP endpoint connected
- authentication completed
- token stored or referenced correctly
- MCP connection verified

## Structure discovery

- default workspace identified
- default space identified
- general to-do destination identified
- ideas destination identified
- project-routing pattern identified

## Defaults

- default assignee identified
- general to-do due-date rule identified
- idea assign/due-date rule identified
- active status vocabulary identified

## Language mapping

- "task list" meaning captured
- "ideas" meaning captured
- project shorthand captured
- client shorthand captured if relevant

## Validation

- safe smoke test run
- results reviewed
- quirks documented

## Ready state

- config saved
- routing rules confirmed
- onboarding summary delivered to user
